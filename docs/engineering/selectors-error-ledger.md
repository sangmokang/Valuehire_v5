## 교훈(원칙으로 승격 후보)
- **0건 = "후보 없음"이 아니라 "흐름/필드 의심" 먼저.** 필드 되읽기 일치 + 결과 0이면 **검색 종류(전역 vs 인재검색)·필드 위치**부터 확인. (W4-3 라이브 인수기준에 반영)
- LinkedIn은 `li.protechts.net ...uc=scraping` 비콘 상시 → 자동 검색 최소화·사람 페이스.
